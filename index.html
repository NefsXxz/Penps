Langkah-langkah

  • 1
    Download File Host
    Klik tombol "Download Host .txt" di samping. File akan otomatis terunduh.
  • 2
    Buka Aplikasi Virtual Host
    Install dan buka aplikasi Virtual Host yang sudah diunduh (HostGo / Virtual Host App).
  • 3
    Pilih File Host
    Tekan tombol "re-select file" atau pilih file, lalu cari file PenPS-Hosts.txt yang baru diunduh.
  • 4
    Aktifkan Host
    Tekan tombol START/ikon power untuk mengaktifkan virtual host.
  • 5
    Mainkan Growtopia
    Buka game Growtopia dan login. Selamat bermain di PenPS!
